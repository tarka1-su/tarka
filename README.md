# منصة إدارة التركة — آل ناصر السلمي

منصة إلكترونية متكاملة لإدارة تركة المرحوم / ناصر محمد السلمي

## المميزات
- إدارة الأصول (نقد، عقارات، أسهم، شركات، وقف)
- نظام المراحل الثلاث (حصر → تأسيس → تسييل)
- القسمة الشرعية الآلية
- مشاريع التسييل والتوزيع
- لجنة الاعتماد والتصويت
- مزامنة فورية عبر Firebase

## الاستخدام
افتح الرابط واختر اسمك من قائمة الورثة.
