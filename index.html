إعداد Firebase

اتبع الخطوات التالية لربط المنصة بقاعدة البيانات

① إنشاء مشروع Firebase
  1. افتح console.firebase.google.com
  2. اضغط + Add project → اختر اسماً (مثل: tarka-alsulmi)
  3. أوقف Google Analytics (غير مطلوب) → Create project
② تفعيل قاعدة البيانات
  1. من القائمة اليسرى: Realtime Database
  2. اضغط Create Database
  3. اختر أقرب منطقة (مثل: europe-west1 أو us-central1)
  4. اختر Start in test mode → Enable
③ تفعيل المصادقة المجهولة
  1. من القائمة: Authentication → Get started
  2. Sign-in method → Anonymous → Enable → Save
④ الحصول على بيانات الاتصال
  1. Project Settings (⚙️) → General → Your apps
  2. اضغط أيقونة الويب </>
  3. اختر اسماً → Register app
  4. انسخ firebaseConfig كاملاً
